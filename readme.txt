#JS API Dashboard

This is a single page website which uses JavaScript, HTML, and CSS to create an API dashboard covering API usage with 8 different API apps. All of the API apps utilize HTTPS and do not require an authorization via apiKey with the exception of the "Trending Movies" card(Instructions Below).

Options to view the website:

A - Using Git 1 - Clone the repo on your local machine 2 - Open git repo in Visual Studio Code on your machine and use the "Live Preview" extension to view

B - Copy Code Manually from Git 1 - Create a new blank project in your visual studio code 2 - Copy the index.html code into a new blank html file in Visual Studio Code and save. 3 - Copy the CSS page into a new styles.css file in Visual Studio Code and save. 4 - Use the "Live Preview" extension in Visual Studio Code to view the webpage

Trending Movies card functionality will not work unless you get an apiKey. To access the movies https://api.themoviedb.org please create a config.js file based on the example file and populate with your TDMB API key to sucessfully use the "Get Movies" button.
